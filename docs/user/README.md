# User Guide - Midway Cleaning Dashboard

## Overview
The Midway Cleaning Dashboard is a comprehensive management system for cleaning service operations. This guide will help you navigate and utilize all features effectively.

## Getting Started

### Accessing the Dashboard
1. Open your web browser
2. Navigate to: https://dashboard.midwaycleaning.com
3. Log in with your credentials

### First-time Setup
1. Change your password
2. Complete your profile
3. Set notification preferences
4. Review company policies

## Dashboard Features

### Home Dashboard
![Dashboard Overview](../assets/dashboard.png)

The main dashboard displays:
- Today's appointments
- Recent customer activity
- Performance metrics
- Important notifications

### Customer Management

#### Adding a New Customer
1. Click "Customers" in the sidebar
2. Click "Add New Customer"
3. Fill in required information:
   - Name
   - Contact details
   - Service address
   - Billing information
4. Click "Save"

#### Managing Customer Information
- View customer details
- Update contact information
- Review service history
- Manage billing preferences

### Appointment Scheduling

#### Creating Appointments
1. Click "Schedule" in the sidebar
2. Select "New Appointment"
3. Choose:
   - Customer
   - Service type
   - Date and time
   - Special instructions
4. Click "Schedule"

#### Managing Appointments
- View daily/weekly/monthly calendar
- Drag and drop to reschedule
- Color-coded status indicators
- Filter by service type or staff

### Service Management

#### Available Services
- Regular Cleaning
- Deep Cleaning
- Move-in/Move-out
- Commercial Cleaning
- Special Requests

#### Customizing Services
1. Select service type
2. Modify included items
3. Adjust pricing
4. Save changes

### Staff Management

#### Assigning Staff
1. Open appointment details
2. Click "Assign Staff"
3. Select available staff members
4. Save assignment

#### Staff Schedule View
- Weekly availability
- Current assignments
- Time-off requests
- Performance metrics

### Billing and Payments

#### Processing Payments
1. Select customer
2. Choose service
3. Enter payment details
4. Process transaction
5. Send receipt

#### Payment Methods
- Credit/Debit Cards
- Bank Transfer
- Digital Wallets
- Invoicing

### Reports and Analytics

#### Generating Reports
1. Click "Reports" in sidebar
2. Select report type:
   - Revenue
   - Customer satisfaction
   - Staff performance
   - Service analytics
3. Choose date range
4. Generate report

#### Customizing Reports
- Filter by date
- Select metrics
- Choose visualization
- Export options

### Notifications

#### Setting Up Notifications
1. Go to "Settings"
2. Select "Notifications"
3. Choose notification types:
   - Appointment reminders
   - Payment confirmations
   - Customer feedback
   - Staff updates
4. Set delivery method:
   - Email
   - SMS
   - In-app
   - Push notifications
